# Customer Lifetime Value Prediction and Customer Segmentation

## Project Overview

This project analyzes customer purchasing behavior using the Online Retail dataset and develops a machine learning pipeline to predict Customer Lifetime Value (CLV).

The objective is to identify valuable customers, forecast future revenue, and generate actionable business recommendations through customer segmentation.

The project combines traditional RFM analysis with predictive modeling and business-focused customer segmentation.

---

## Dataset

**Online Retail Dataset**

The dataset contains transactional records from a UK-based online retailer, including:

* Invoice Number
* Product Information
* Quantity Purchased
* Unit Price
* Invoice Date
* Customer ID
* Country

---

## Project Workflow

### 1. Data Cleaning

The following preprocessing steps were performed:

* Removed transactions with missing Customer IDs
* Removed cancelled invoices
* Removed transactions with negative quantities
* Removed transactions with non-positive prices
* Converted InvoiceDate to datetime format
* Created a Revenue feature

Revenue was calculated as:

Revenue = Quantity × UnitPrice

---

### 2. Time-Based CLV Framework

To avoid data leakage and simulate a real-world prediction scenario, the dataset was divided into two periods:

#### Past Window

Used to generate customer features.

#### Future Window

Used to calculate actual future customer value (target variable).

This allows the model to learn historical purchasing behavior and predict future spending.

---

### 3. Feature Engineering

Customer-level features were created from the Past Window.

#### Recency

Number of days since the customer's most recent purchase.

#### Frequency

Number of unique purchases made by the customer.

#### Monetary Value

Total revenue generated by the customer.

#### Average Order Value (AOV)

Average amount spent per order.

#### Customer Tenure

Number of days between the customer's first and last purchase.

---

### 4. Target Variable

Future Customer Lifetime Value (FutureCLV) was calculated as:

FutureCLV = Total Customer Revenue in the Future Window

Customers with no purchases during the Future Window were assigned a value of 0.

---

### 5. Target Transformation

Because customer lifetime value is highly skewed, a logarithmic transformation was applied:

FutureCLV_Log = log(1 + FutureCLV)

This reduces the impact of extreme spending values and improves model stability.

---

### 6. Machine Learning Models

Two regression models were evaluated:

#### Linear Regression

Used as a baseline model.

#### Random Forest Regressor

Used as the primary predictive model due to its ability to capture non-linear customer behavior patterns.

---

## Model Performance

| Model             | MAE  | R²   |
| ----------------- | ---- | ---- |
| Linear Regression | 2.57 | 0.19 |
| Random Forest     | 2.44 | 0.16 |

### Key Findings

* Random Forest achieved a slightly lower prediction error (MAE).
* Linear Regression achieved a slightly higher R² score.
* Historical customer spending behavior provides meaningful signals for predicting future value.
* Customer lifetime value remains inherently difficult to predict due to behavioral variability.

---

## Feature Importance Analysis

Random Forest feature importance revealed:

| Feature             | Importance |
| ------------------- | ---------- |
| Monetary Value      | 45.3%      |
| Recency             | 24.2%      |
| Average Order Value | 19.4%      |
| Tenure Days         | 7.8%       |
| Frequency           | 3.2%       |

### Insight

Monetary Value was the strongest predictor of future customer value, followed by Recency and Average Order Value.

This suggests that how much customers spend is a stronger predictor of future revenue than how frequently they purchase.

---

## Customer Retention Analysis

Approximately **67.3%** of customers returned and made purchases during the Future Window.

Approximately **32.7%** of customers generated no future revenue.

This indicates a relatively strong customer retention rate within the analyzed period.

## Customer Segmentation

Customers were segmented using predicted CLV scores into four groups:

* Low Value
* Mid Value
* High Value
* VIP
